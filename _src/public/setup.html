About Lux

Lux is a Philadelphia Game Lab project focused on providing Backend Services to games and applications. The driving factor behind Lux has always been a desire to not limit the functionality or creativity of any application developed on the Lux system, while still providing a system that would be useful for developers. Although Lux is growing everyday, and will continue to develop and grow, the project has reached the point where we can start to develop applications on Lux.

Lux allows developers to focus on developing their application, and not managing database or HTTP functionality

Lux uses your information and your application to give you analytics with a purpose.

Lux moves your entire application to the cloud. An AWS cloud to be exact, making sure your application's servers never go down.

About The Demo Game

Rather than spend hours talking about Lux, it seemed appropriate that we should show you Lux in action. We built this game purely in JS, and gave it multiplayer functionality using Lux- without writing a single custom piece of server-side code. The Lux team thought that a Mutli-player game was the best way to show how Lux can be useful to you, but we also included a few extra elements, for instance the authorization is handled by Lux as well. The game won't be winning any awards by it's self- but watching Lux in action is what this game is really all about.

First we developed the game and the art without any back-end or multiplayer functionality, purely a single player game.

Then we used google's OAuth (which is built into Lux) as the player's identifier for Lux's matchmaker API.

Finally we integrated Lux into the game, which allowed us to take this single player game to a multiplayer state using only Lux.

Instructions on How to Play

This game has a very simple premise- capture as many flags as possible. Each player starts with 2 flags, and the winner is determined by who has the most flags when time runs out. To run, use the [a,s,z,x] keys. To capture a flag, use the [c] key. In order to attack your opponent (the easiest way to defend your flag), face them and hit spacebar. You'll need to get pretty close to capture a flag, but even closer to attack an opponent.
